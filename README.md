# BuffSteamExchange

BuffSteamExchange 是一个连接 Buff163 市场和 Steam 社区市场的全自动交易与数据分析平台。本系统提供智能的市场扫描、算法驱动的购买推荐、全自动的 Steam 库存管理，以及一个用于管理完整交易生命周期的可视化 Web 控制面板。

## 核心功能

### 1. 算法驱动的市场分析
- 基于交易量和用户自定义的价格区间自动扫描 Buff163 市场饰品。
- 采用量化指标（SMA 移动平均、RSI 相对强弱指数、R-Squared 趋势斜率）对 CSGO/CS2 饰品的购买价值进行精准评估。
- 结合近期历史价格波动曲线，智能识别最佳买入时机。

### 2. 全生命周期自动交易
- **自动购买:** 使用用户配置的 Cookie，在 Buff163 上实现无人值守的自动化挂单与购买。
- **Steam 自动收货:** 深度集成 `steamcommunity` 和 `steam-session` 库，在检测到报价后自动确认并接收已购买的饰品。
- **自动上架售卖:** 定时扫描您的 Steam 库存，一旦饰品冷却期（CD）结束，系统将根据设定的价格策略自动将其挂至 Steam 社区市场出售。

### 3. 多功能 Web 控制面板
- **市场看板:** 实时展示 Buff 饰品的核心分析数据，辅以直观的颜色标识指导购买决策。
- **库存管理:** 对“待收货”、“已在库”和“待出售”状态的饰品进行全流程追踪。
- **人工干预:** 允许用户在系统中手动添加、绑定或删除追踪物品。
- **利润统计:** 详细记录交易历史、总支出、预期回血金额以及平均折扣率。

### 4. 健壮的配置与安全机制
- **可视化配置:** 可直接在前端设置界面安全地更新 Steam 账号密码、Buff Cookie、网络代理以及 Webhook 报警地址。
- **安全验证:** 控制面板内置 Basic Auth 基础认证，并提供基于 Cookie 的持久化免密登录支持。
- **熔断与报警:** 当系统检测到 Cookie 失效或 API 请求受到限制时，会自动挂起抓取任务并通过 Webhook 及时推送通知。

## 安装指南

1. **环境准备**
   请确保您的操作系统已安装 [Node.js](https://nodejs.org/)。

2. **获取源码**
   下载或克隆本项目的完整代码目录至本地。

3. **安装依赖**
   在项目根目录下打开命令行工具并运行：
   ```bash
   npm install
   ```

## 系统配置

系统提供两种配置途径：

1. **通过 `.env` 配置文件**
   将根目录下的 `.env.example` 复制并重命名为 `.env`，然后填入以下参数：
   - `BuffCookie`: 您的 Buff163 账号登录 Cookie。
   - `SteamAccount` / `SteamPassword`: Base64 编码后的 Steam 账号和密码。
   - `SharedSecret` / `IdentitySecret`: 用于生成 2FA 令牌及自动确认交易报价的密钥。
   - `ProxyUrl`: Steam 网络请求的代理地址（如果在国内网络环境下必须配置）。
   - `BuffMinPrice` / `BuffMaxPrice`: 市场扫描的价格上下限过滤规则。

2. **通过 Web 控制面板**
   您也可以直接启动程序，在控制面板的“系统设置”页面中通过可视化界面配置所有敏感信息。

## 启动与使用

在命令行中输入以下命令启动服务：
```bash
npm start
```

当服务成功启动后，请在浏览器中访问：
```
http://localhost:9998
```
*注：如果您未在设置中修改，系统的默认登录账号为 `admin`，密码为 `123456`。*

## 定时任务与后台调度进程

系统在后台运行着一个全自动的调度守护进程 (`server/scheduler.js`)，它负责：
- 周期性抓取和更新 Buff 饰品数据源，确保量化分析建立在最新数据之上（超过 24 小时未更新会自动触发强制刷新）。
- 定期扫描您的 Steam 账号库存，实时同步已购入饰品的最新状态。
- 每隔 24 小时对底层库存进行盘点，并自动将所有可交易的饰品挂至 Steam 市场。
- 每隔 5 分钟执行一次轻量级的轮询，用于处理待确认的收货请求与挂单异常。
